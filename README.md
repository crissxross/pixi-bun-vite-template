# PixiJS Bun Vite TS Template


Clone this repo & start developing!

Pixi version 7 & vanilla TypeScript

## get started

```
$ cd your-repo-name
$ bun install
$ bun run dev
```

## build

```
$ bunx --bun vite build
```
