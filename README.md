# PixiJS Bun Vite TS Template


Clone this repo & start developing! (Note: WIP)

![Pixi JS logo](/public/logo/pixi.svg)

Pixi v.7, GSAP v.3 & vanilla TypeScript

## get started

```
$ cd your-repo-name
$ bun install
$ bun run dev
```

## build

```
$ bunx --bun vite build
```
